CDK - Chris Development Kit
Ferramentas úteis para agilizar no desenvolvimento de mecânicas principais na Unity.

Esse código é compartilhado entre todos os meus projetos e está em ativo desenvolvimento!

Discord Bot: Nyu
Nyu é um bot que fiz para o Discord, tem funções de administrativas, comunicação com APIs externas, banco de dados local e roda num servidor na GCP, usado principalmente no meu servidor.

O bot tem além de funções úteis algumas funções de entretenimento como memes, recordar mensagens antigas e tocar áudios. Você pode ver o código fonte dele no Github.

  • Face 3D - Mobile App

    A View to a Kill (powered by Tobii Eye Tracking technology)

     Comédia Acessibilidade
    Projeto feito para jogar usando o Tobii Eye Tracking (controle usando os olhos).

    Tobii Eye Tracking é uma camera que lê com precisão onde na tela a pessoa esta olhando, essa informação pode e foi usada no jogo. No jogo seu personagem tem um super poder de soltar laser pelos olhos, o objetivo é derrotar os inimigos olhando para eles mas não olhando para civis.
